# PlatinumRx Data Analyst Assignment

## 📌 Overview

This project demonstrates core data analysis skills including:

* SQL (Database Management & Querying)
* Excel (Data Analysis & Lookup Functions)
* Python (Programming Logic)

## 🏨 SQL Tasks

### Hotel System

* Created tables for users, bookings, items, and billing
* Performed:

  * Latest booking per user
  * Monthly billing calculations
  * High-value bill filtering
  * Item popularity analysis using window functions
  * Ranking-based queries

### Clinic System

* Created tables for clinics, sales, and expenses
* Performed:

  * Revenue by channel
  * Customer ranking
  * Monthly profit analysis
  * City/state profitability analysis

---

## 📊 Spreadsheet Tasks

* Used **VLOOKUP / INDEX-MATCH** to merge data
* Performed **same day & same hour analysis**
* Used helper columns and logical functions

## 🐍 Python Tasks
